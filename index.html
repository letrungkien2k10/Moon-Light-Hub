Frequently Asked Questions

Answers to common questions about scripts and the community

How do I execute (run) a script?

You need to download a Roblox Executor such as Synapse X, Krnl, Fluxus, or Delta. Then copy the script from the website, paste it into the executor and press Execute/Inject. Each executor works differently — join the Discord server for detailed guidance.

Can using scripts get me banned?

There is a risk of being banned if you use scripts aggressively or in games with strong anti-cheat systems. We recommend using an alt account to test first and not using scripts too obviously. Some scripts have anti-detection features but cannot guarantee 100% safety.

Which executors are supported?

Most scripts on the website are compatible with popular executors: Synapse X (paid), Krnl (free), Fluxus (free), Delta (free), and Codex (free). If a script does not run on your executor, try updating it to the latest version or contact Discord for support.

How does the key system work?

Some scripts require a key to use. Keys can be obtained for free via Linkvertise or other link shortener pages. After completing the link, you will receive a key valid for 24–48 hours. Keys help the developer earn income to maintain and update scripts. Join Discord to get your key faster!

What should I do if a script is broken or won't run?

First make sure your executor is up to date. If it still fails, try these steps: 1) Rejoin a new server, 2) Run the script at the main screen before entering the game, 3) Check if the game has received a new anti-cheat patch. If it still doesn't work, report the bug in the #bug-report channel on Discord for the admin to fix.
