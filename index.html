Giải đáp thắc mắc phổ biến về script và cộng đồng

Làm sao để execute (chạy) script?

Bạn cần tải về một Roblox Executor như Synapse X, Krnl, Fluxus, hoặc Delta. Sau đó copy script từ website, dán vào executor và nhấn Execute/Inject. Mỗi executor có cách sử dụng khác nhau, bạn có thể vào Discord server để được hướng dẫn chi tiết.

Sử dụng script có bị ban (band) không?

Có rủi ro bị ban nếu sử dụng script một cách thô bạo hoặc trong các game có hệ thống anti-cheat mạnh. Chúng tôi khuyên nên sử dụng tài khoản phụ (alt account) để test trước, và không sử dụng script quá lộ liễu. Một số script có tính năng anti-detection nhưng không đảm bảo 100%.

Script này hỗ trợ executor nào?

Hầu hết script trên website đều tương thích với các executor phổ biến: Synapse X (paid), Krnl (free), Fluxus (free), Delta (free), và Codex (free). Nếu script không chạy trên executor của bạn, hãy thử cập nhật executor lên phiên bản mới nhất hoặc liên hệ Discord để được hỗ trợ.

Key system hoạt động thế nào?

Một số script yêu cầu key để sử dụng. Key có thể lấy miễn phí qua linkvertise hoặc các trang rút gọn link. Sau khi vượt link, bạn sẽ nhận được key có hiệu lực trong 24h-48h. Key giúp developer có thu nhập để duy trì và cập nhật script. Hãy vào Discord để nhận key nhanh hơn!

Script bị lỗi không chạy phải làm sao?

Đầu tiên hãy đảm bảo executor của bạn đã được cập nhật. Nếu vẫn lỗi, hãy thử các bước: 1) Rejoin server mới, 2) Chạy script ở màn hình chính trước khi vào game, 3) Kiểm tra xem game có bị cập nhật patch chống hack mới không. Nếu vẫn không được, báo lỗi trong kênh #bug-report trên Discord để admin fix.
